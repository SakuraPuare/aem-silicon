# aem-silicon

本仓库为从百度 Apollo 的 AEM 相关脚本与资源（打包于 aem deb/发布归档中）提取与整理的独立仓库，目标是在 Apple Silicon (arm64, macOS) 上进行适配与迁移，便于在 Apple 芯片设备上进行环境管理与开发体验改进。

- 来源：基于 Apollo 项目的 AEM 组件与脚本提取与重构
- 目的：在 Apple Silicon 平台上最小变更实现可用，并逐步完善兼容性
- 状态：进行中，欢迎问题反馈与贡献 PR
